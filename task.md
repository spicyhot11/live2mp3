# 程序说明

1. 将批量视频自动转换为MP3格式（TS\MP4）
2. 转换的视频所在目录，要求可以配置两个部分：1. 根目录  2. 相对该根目录的子路径。 子路径支持 黑白名单以及通配
3. 支持为每个路径配置是否保留原始文件
4. 输出mp3格式文件，要求如下：
   1. 支持为每个视频路径 配置对应mp3输出路径。 默认为当前视频所在的文件夹名
   2. 自动检查视频与MP3格式是否正确，不正确尝试修复，修复失败即删除
   3. 自动拼接mp3文件，按照MP3文件名所表明的时间进行拼接。拼接后以最早时间作为文件名，并删除原始文件，相邻文件相差N小时(可配置)内的即视为同一个MP3。文件名格式有以下两种：
      1. `[2026-01-06 09-47-38][Neet仓鼠鼠][保密工作进行中]`
      2. `录制-7391879-20240801-151938-265-和木绵绵一起直播看日记`
      3. 可配置定时任务
5. 提供简单的前后端。  实体机是mac arm  m4。 后端由C++实现，前端可采用vue，使用docker容器。
